    # 1.2.3 事件传参

    1. 不能在绑定事件的同时，为事件处理函数传递参数
    2. 可以为组件提供 data-* 自定义属性传参 其中 * 是参数的名字
    3. 通过 e.target.dataset.参数名 可以获取到具体参数信息
    4. data-info='2' 传递的是字符串 "2"
    5. data-info=2 传递的是数字 2
        // 一点击图片改变；count与 组件传递的参数 相加
    btnTap(e){
        this.setData({
            imgSrc:'/images/3.jpg',
            count:this.data.count+e.target.dataset.info
        })
        console.log(e.target.dataset.info)
        console.log(this.data.count)

    1. wx:if 与 hidden 之间的区别

    运行方式不同

    wx:if 以动态创建与移除元素的方式，控制元素的展示与隐藏 hidden 以切换样式的方式『display:none/block』, 来控制元素的显示与隐藏

    使用建议

    频繁切换时，最好不要频繁操作dom 建议使用 hidden 控制条件复杂时，建议使用 wx:if 搭配 wx:elif、wx:else 来进行切换

</view>

    1. 使用wx:for-index 可以指定当前循环项的所有的变量名
    2. 使用wx:for-item 可以指定当前循环项的变量名

    # 2. wxss模板样式

    wxss 比 css 扩展的属性

    1. rpx 尺寸单位 : 用来解决屏适配的尺寸单位

    原因： 在宽度上等分为750份 『即当前屏幕的总宽度是750rpx』 在较大的设备上，1rpx所代表的宽度较大 在较小的设备上，1rpx所代表的宽度较小

    1. 导入样式：@import
    @import 'common.wxss'
    1
    1. 全局样式：定义在app.wxss
    2. 局部样式：写在页面上的wxss，只作用与当前页面

    注意：

    1. 权重比：局部样式 > 全局样式 『局部样式会覆盖全局样式』
    2. 权重较小就无法覆盖

    # 4. tabBar

    # 4.1 概念

    tabBar 是移动端应用常见的页面效果，用于实现多页面的快速切换。

    # 4.2 组成部分

    • backgroundColor：tabBar 的背景色
    • selectedIconPath：选中时的图片路径
    • borderStyle：tabBar 上边框的颜色
    • iconPath：未选中时的图片路径
    • selectedColor：tab 上的文字选中时的颜色
    • color：tab上文字的默认（未选中）颜色

    # 4.3 配置

    1. 必填项：list『是Array类型』
        "tabBar": {
      "list": [{
          "pagePath": "pages/index/index",
          "text": "index"
        },
        {
          "pagePath": "pages/list1/list1",
          "text": "list"
        }
      ]
    }

    # 6. 网络数据请求

    # 6.1 限制

    1. 只能请求 HTTPS 类型的接口
    2. 必须将接口的域名添加到信任列表中
    3. 步骤：详情 ==> 项目配置 ==> 合法域名

    # 6.2 配置合法域名

    1. 配置步骤：
    • 登录微信小程序管理后台 ==> 开发 ==> 开发设置 ==> 服务器域名 ==> 修改request合法域名
    1. 注意：
    • 域名只支持 https 协议
    • 域名不能使用ip地址 包括localhost
    • 域名必须经过ICP备案
    • 服务器域名一个月内最多可以申请5次修改

    1. 跳过request合法域名校验

      步骤： 微信开发者工具中，临时开启「开发环境不校验请求域名」选项，跳过request 合法域名的校验 注意： 该选项仅限在开发与调试阶段使用

    # 6.3 发起请求

    1. GET请求：使用wx.request()
        // 发起GET请求
    getInfo(){
        wx.request({
            // 接口地址:必须基于https协议
            url: 'https://www.escook.cn/api/get',
            method:'GET',
            // 发送到服务器的数据
            data:{
                name:'zs',
                age:20
            },
            // 请求成功后的回调函数
            success:(res)=>{
                console.log(res.data)
            }
        })
    },
