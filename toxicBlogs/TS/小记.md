## 1. TypeScript 函数重载

同一个函数名可以有多个签名，TS会根据调用时传入的参数类型自动选择最匹配的版本。

TS会从上到下找第一个匹配的重载，所以重载声明的顺序很重要（具体的放前面，宽泛的放后面）

例子：

```ts
  // 重载1：只传命令，使用默认选项                                                                             
  function exec(                                                                                             
      command: string,                                                                                       
      callback?: (error: ExecException | null, stdout: string, stderr: string) => void,                       
  ): ChildProcess;                                                                                           
   
  // 重载3：传 string 编码选项                                                                                 
  function exec(  
      command: string,                                                                                       
      options: ExecOptionsWithStringEncoding,
      callback?: (error: ExecException | null, stdout: string, stderr: string) => void,
  ): ChildProcess;                                                                                                                                                                                                         
  // 实际实现（唯一真实函数体）                                                                                   
  function exec(  
      command: string,
      options?: ExecOptions | null,                                                                           
      callback?: (error: ExecException | null, stdout: string | NonSharedBuffer, stderr: string | NonSharedBuffer) => void,
  ): ChildProcess {                                                                                           
  }                         
```

```bash
  # 调用时匹配规则                                                                                            
  exec("ls", (err, stdout) => {});                    # 匹配重载1                                            
  exec("ls", { encoding: "utf8" }, (err, str) => {});   # 匹配重载3
  exec("ls", { cwd: process.cwd(), timeout: 120000 }, (err, stdout, stderr) => {}) # 实际使用
```
